## Background

I work at the intersection of research and engineering.
My interests span AI, agent systems, scientific computing, circuit design, quantum computing, and quantum memory, with a particular focus on turning ideas into usable tools, simulation workflows, and real technical systems.

## Representative Projects

- [actoviq-agent-sdk](https://github.com/DeconBear/actoviq-agent-sdk)  
  An independent and experimental agent SDK for practical multi-tool, multi-session, and bridge-assisted workflows. The project explores how agent systems can become usable developer infrastructure rather than isolated demos, with support for sessions, tools, MCP integration, workspace helpers, runtime bridges, and interactive streaming workflows. It reflects much of my interest in agent runtime design, orchestration, and developer ergonomics.

- [SmithTool](https://github.com/DeconBear/SmithTool)  
  An interactive Smith Chart desktop application for RF impedance matching and matching-network design, built with C++17 and Qt 6 Widgets. It includes impedance editing, matching element design, Touchstone import, SPICE netlist export, and bilingual UI support. This project is closely connected to my interests in circuit design, engineering software, and practical tools for real technical work.

- [klynx_skills](https://github.com/DeconBear/klynx_skills)  
  A reusable skills repository for engineering workflows, currently centered on circuit design and simulation with ngspice. It provides a simulation-feedback loop including netlist validation, batch execution, metric extraction, report export, and schematic rendering. The repository reflects my interest in combining AI-assisted workflows with circuit analysis and computational design tooling.

## Participated Projects

- [QuTiP](https://github.com/qutip/qutip)  
  An open-source framework for simulating quantum systems, widely used in quantum optics, quantum information, and related computational physics.

- [QuTiP-qip](https://github.com/qutip/qutip-qip)  
  A QuTiP ecosystem project focused on quantum information processing, including circuit-level simulation and related workflow support.
